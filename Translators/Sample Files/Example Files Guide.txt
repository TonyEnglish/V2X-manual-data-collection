Example Files Guide

CAMP_RSM_example.xml is an xml RSM file generated from the CAMP tool
CAMP_WZDx_example.geojson is a geojson WZDx file generated from the CAMP tool (generated alongside CAMP_RSM_example.xml)
CAMP_RSM_example.uper is CAMP_RSM_example.xml converted to UPER format using the message builder application
CAMP_RSM_example_from_uper.xml CAMP_RSM_example.uper converted back to XML format
CAMP_WZDx_translated_example.geojson is a geojson WZDx file translated from CAMP_RSM_example.xml
WZDX_linestring_example.geojson is a geojson WZDx sample file from https://github.com/usdot-jpo-ode/jpo-wzdx

CAMP_RSM_example.xml -> RSM to WZDx Translator -> CAMP_WZDx_translated_example.geojson
CAMP_RSM_example.xml -> Message Builder XmltoUper -> CAMP_RSM_example.uper -> Message Builder UpertoXml -> CAMP_RSM_example_from_uper.xml
